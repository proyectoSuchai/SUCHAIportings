Documentación de comandos para el satélite SUCHAI

Navegue por los distintos archivos y secciones para encontrar el código y documentación correspondiente a todos los comandos disponibles en el sistema de vuelo. También puede utilizar el buscador integrados.

Cada comando se representa a través de un código en hexadecimal y recibe un argumento entero.

Los comandos incluidos pueden ser ejecutados por consola a través de la orden

exe_cmd <comando> <argumento>

o a través de telecomandos desde la estación terrena, enviando frames en formato hexadecimal, bigendian donde cada comando y argumento corresponde a 16bits. Se usa la señal de termino especial 0xFFFE para marcar el final del frame.

<cmd><arg><cmd><arg> ······ <stop>

Comandos disponibles

  • Comandos de consola : CON_CmdIndx
  • Comandos para repositorio de datos : DRP_CmdIndx
  • Comandos para sistema de energía : EPS_CmdIndx
  • Comandos para payloads : PAY_CmdIndx
  • Comandos para PIC-PC104 : PPC_CmdIndx
  • Comandos para RTC : RTC_CmdIndx
  • Comandos para telemetría y telecomandos : TCM_CmdIndx
  • Comandos para TRX : TRX_CmdIndx
